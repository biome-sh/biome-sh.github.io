Overview

  • Self-contained application builds

    Each build of your application is self-contained and thus self-sufficient. All dependencies are specified explicitly, ensuring that on deployment it runs exactly as designed, on the exact versions of dependencies you specify. Bypassing Operating System dependencies for guaranteed success.

  • Deployment channels and management

    Manage your deployment with channels via Biome. Allow automatic updates on your deployed nodes when a pacakge is promoted to a new channel. Simplify your deployment drastically.

  • Deploy Anywhere

    Applications packaged with Biome are portable. Once built, they can be exported for almost any runtime available, including Docker, Kubernetes, Mesos, VMs, Bare Metal.. you name it.
